# HTML

html 태그

* h1 ~ h6 : 제목(heading) 태그
* p : 단락(paragraph) 태그
* br : 줄바꿈(linebreak) 태그
* b : 글자 굵게
* i : 글자 이탤릭체
* u : 글자에 밑줄 
* s : 글자에 중간선
* a : 다른 문서로 이동할 수 있는 링크 생성 
    * href : "링크 url"
    * targrt : _self(현재 페이지에 표시), _blank(새로운 페이지에 표시)
* 의미없이 요소를 묶기 위한 태그(Container)
    * div : block-level
    * span : inline-level
